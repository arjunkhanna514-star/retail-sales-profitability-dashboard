RETAIL SALES & PROFITABILITY ANALYSIS DASHBOARD
 
OVERVIEW

This project is an Excel-based financial dashboard analyzing retail revenue, cost structure, and profitability trends over 5 years.

 KEY FEATURES
 
Revenue & Profit Analysis

Profit Margin Calculation

Year-over-Year Growth

Revenue Forecast Model

Professional Data Visualization

 TOOL USED

Microsoft Excel

Financial Ratio Analysis

Forecasting Techniques

OBJECTIVE
 
  To simulate business performance evaluation and support data-driven decision-making.
